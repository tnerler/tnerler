## About Me

I'm a Computer Engineering student at **Piri Reis University**, focused on building practical machine learning systems end-to-end — from data pipelines and model training to deployment and explainability.

My work centers on a few core areas:

- Machine Learning Engineering
- Artificial Intelligence
- Large Language Models
- Retrieval-Augmented Generation (RAG)
- Recommender Systems
- Computer Vision
- AI for Games

I'm currently deepening my knowledge of **MLOps**, **Transformers**, **LLM Applications**, and **end-to-end Machine Learning systems** — the kind of infrastructure and design decisions that separate a notebook experiment from a production model.
